# 🚀 InterviewAI Pro  
**Intelligent Interview Preparation System**

## 📌 Overview
InterviewAI Pro is an AI-powered web application designed to help users prepare for job interviews efficiently. It simulates real interview scenarios, analyzes user responses, and provides feedback to improve performance.

---

## ✨ Features
- 🎤 AI-Based Mock Interviews  
- 🧠 Intelligent Feedback System  
- 📊 Performance Analysis  
- 🌐 User-Friendly Interface  
- 📁 Multiple Domains Support  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, EJS  
- Backend: Node.js, Express.js  
- Database: MongoDB
